PdM/シニアエンジニアへの確認事項を整理してください。

以下を実施してください:
1. 仕様の不明点をリストアップ
2. 技術判断が必要な点をリストアップ
3. 各項目について確認先を明示（PdM/Senior Eng/Designer等）
4. 質問テンプレートを提示

ユーザーが質問を投稿したら、以下を確認してください:
- GitHub Issue/CommentのURL、またはSlackスレッドのURL
- 確認先（PdM/Senior Eng等）

その後、TodoWriteツールで質問タスクを作成してください:
```
{
  content: "[確認先]確認: [確認内容の要約]",
  activeForm: "[確認先]からの回答待ち",
  status: "pending"
}
```

質問のリンクと確認先を会話内に明記してください。

完了後:
- 現在のフェーズを "clarifying" に設定
- 確認事項がない場合は、次のステップとして `/task_implement` を提案してください
- 確認事項がある場合は、回答を待ってから `/task_implement` に進むよう案内してください
