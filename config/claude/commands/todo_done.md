指定された番号のタスクを完了にしてください。

引数として番号を受け取ります（例: `/todo_done 3`）

手順:
1. 指定された番号のタスクを特定
2. 質問タスクの場合、回答内容を確認
   - 「回答内容を教えてください（スキップする場合はEnter）」と質問
   - 入力された場合、回答内容を記録
3. TodoWriteツールでステータスをcompletedに更新
4. 完了を確認するメッセージを表示

例（実装タスク）:
```
✅ タスク #3 を完了にしました
「RealTimeReservationModalのスタイル調整」
```

例（質問タスク）:
```
回答内容を教えてください（スキップする場合はEnter）:
> モーダルは即座に表示でOK

✅ 質問タスク #4 を完了にしました
「PdM確認: モーダル表示タイミング」
💬 回答: モーダルは即座に表示でOK
```
