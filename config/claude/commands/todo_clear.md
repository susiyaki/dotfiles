現在のTODOリストを全てクリアします。

手順:
1. 現在のTODOリストを表示
2. 確認メッセージを表示
   - 「現在のTODOリスト（[件数]件）をクリアしますか？ (y/n)」
3. yの場合、TodoWriteツールで空のリストを設定
4. クリア完了のメッセージを表示

例:
```
📋 現在のTODOリスト（3件）:
1. ⏳ RealTimeReservationModalのスタイル調整
2. ⏳ Provider設置場所を移動
3. ⏳ PdM確認: モーダル表示タイミング

このTODOリストをクリアしますか？ (y/n)
> y

✅ TODOリストをクリアしました
```

注意: 質問タスクのリンクや回答情報は会話履歴から削除されませんが、TODOリストからは削除されます。
